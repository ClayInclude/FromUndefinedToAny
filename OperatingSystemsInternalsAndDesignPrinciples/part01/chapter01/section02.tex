\section{微处理器的发展}
{
    微处理器不仅发展为最快的通用处理器，还发展成了多处理器：
    每个\emreg{芯片（插槽，Socket）}上包含多个\emreg{处理器（核心，Core）}，每个处理器配备多级大容量缓存，而且多个逻辑处理器之间共享内核的执行单元。

    \emreg{图形处理单元(Graphical Processing Units, GPU)}能利用单指令多数据流(Single Instruction Multiple Data, SIMD)技术实现数组数据的有效计算。

    \emreg{数字信号处理器(Digital Signal Processors, DSP)}也是主要用于流信号处理的处理器。

    为了满足手持设备的需求，传统的微处理器正让路于\emreg{片上系统(System on a Chip, SoC)}。
    在Soc中，不仅CPU和高速缓存共存于同一个芯片上，系统中的大多数硬件也都被集成到该芯片上。
}
