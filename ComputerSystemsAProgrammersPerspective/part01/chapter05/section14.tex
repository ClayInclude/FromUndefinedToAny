\section{确认和消除性能瓶颈}
{
    \emreg{代码剖析程序(code profiler)}是在程序执行时收集性能数据的分析工具。

    \subsection{程序剖析}
    {
        程序\emreg{剖析(profiling)}运行程序的一个版本，其中插入了工具代码，以确定程序的各个部分需要多少时间。
        剖析的一个有力之处在于可以在现实的\emreg{基准数据(benchmark data)}上运行实际程序的同时，进行剖析。
    }

    \subsection{使用剖析程序来指导优化}
    {
        Amdahl定律为通过有针对性的优化来获取性能提升提供了一些其他的见解。
    }
}
