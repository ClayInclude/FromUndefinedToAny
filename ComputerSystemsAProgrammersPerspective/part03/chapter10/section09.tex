\section{I/O重定向}
{
    Linux shell提供了\emreg{I/O重定向}操作符，允许用户将磁盘文件和标准输入输出联系起来。

    dup2函数复制描述符表表项oldfd到描述符表表项newfd，覆盖描述符表表项newfd以前的内容。

    %4.
    \begin{practicec}
        dup2(5, 0)
    \end{practicec}

    %5.
    \begin{practicec}
        o
    \end{practicec}
}
