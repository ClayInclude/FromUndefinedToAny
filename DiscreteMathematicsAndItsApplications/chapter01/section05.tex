\section{嵌套量词}
{
    \subsection{引言}
    {
        \emreg{嵌套量词}即一个量词出现在另一个量词的作用域内。
    }

    \subsection{理解涉及嵌套量词的语句}
    {
        \paraph{将量化当作循环}
        {
            在处理多个变量的量化式时，有时候借助嵌套循环来思考是有益的。
        }
    }

    \subsection{量词的顺序}
    {
        量词的顺序是很重要的，除非所有量词均为全称量词或均为存在量词。
    }
}
